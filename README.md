# 서비스 소개

<aside>
💡 🌱새싹대학교🌱 LMS 학사관리시스템 서비스입니다. 
새싹대학교의 학생, 교수, 관리자로 나뉘어서 필요한 기능을 만들었습니다.
대표적으로 강의 개설, 수강신청, 시험, 출석, 성적 조회 등등 기능을 구성했고
각각의 기능은 로그인한 계정의 권한에 따라 다르게 작동하도록 구현하였습니다.

</aside>
